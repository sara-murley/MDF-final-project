# Internet Connectivity and Socioeconomic Outcomes

## A Big Data Policy Analytics Project

**Sara Murley · Samantha Rudravajhala · Mena Tetali**  
Georgetown University — PPOL 5206

---

## Overview

This project examines how internet connectivity influences key socioeconomic outcomes across countries, including:

- Income (GDP per capita)
- Education (school enrollment)
- Health (life expectancy)

We build a cloud-based pipeline using:

- World Bank API
- AWS
- Python (analysis + regression modeling)
